# new_project

Makes a directory and skeleton project

## Installation

Add this line to your application's Gemfile:

    gem 'new_project'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install 'new_project'

## Usage

    $ new_project <project_name_in_snake_case>
