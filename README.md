# new_project

Makes a directory and skeleton project

## Installation

    $ git clone git@github.com:kthibodeaux/new_project.git
    $ cd new_project
    $ gem build new_project.gemspec
    $ gem install new_project-x.x.x.gem

## Usage

    $ new_project <project_name_in_snake_case>
